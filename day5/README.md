# Blocksmith

Blocksmith is a tiny time block divider that turns a task list into a schedule with real start and end times.

## Why this exists

It is easy to over-commit when you only have a short window. A quick split into clear blocks keeps you realistic and on pace.

## Features

- Equal or weighted splits using simple task weights
- Optional short breaks between tasks
- Copy-friendly schedule output
- Local-only, no accounts or tracking

## Run

Open `index.html` in a browser.
