# PackPilot

PackPilot is a tiny packing list generator that turns trip type and duration into a checklist you can finish quickly.

## Why this exists

Packing for short trips is easy to overthink. A short, realistic list keeps you from overpacking or forgetting essentials.

## Features

- Trip-specific checklist for city, beach, business, camping, and winter trips
- Clothing counts that adjust to trip length and laundry access
- Optional extras for work, fitness, kids, or camera gear
- Custom items with local-only persistence
- Copyable checklist for notes apps

## Run

Open `index.html` in a browser.
