# Timebridge

Timebridge is a meeting time zone converter that shows everyone their local time in one view.

## Why this exists

Cross-time-zone scheduling usually requires double-checking multiple apps. Timebridge keeps the meeting time anchored and makes it easy to share the right times fast.

## Features

- Set a meeting date/time and pick a source time zone
- Compare up to four additional time zones
- Day-shift labels (same day, next day, previous day)
- Copyable summary line for invites or chat
- Local-only, no accounts

## Run

Open `index.html` in a browser.
